    Introduction

    Welcome to the Jeopardy API! You can use our API to access Jeopardy API endpoints, which can get information on various shows, rounds, and questions in the database.

    Authentication

    There is no authentication for the Jeopardy API but it is rate limited.

    Please use this responsibly.

    Question

    Get A Random Question

    curl "https://jeopardy-api.bentleyherron.dev/api/question"

    The above command returns JSON structured like this:

    {
    "success": true,
    "message": "Got a random show question.",
    "question": {
        "Show Number": 5422,
        "Air Date": "2008-03-18",
        "Round": "Double Jeopardy!",
        "Category": "THE LOVELY BONES",
        "Value": "$1200",
        "Question": "The stapes, the smallest bone in the human body, is found in this organ",
        "Answer": "the ear"
    }
}

    This endpoint retrieves a random question.

    HTTP Request

    GET https://jeopardy-api.bentleyherron.dev/api/question

    This endpoint retrieves a specific show or shows.

    HTTP Request

    GET https://jeopardy-api.bentleyherron.dev/api/shows/<show_number>

    URL Parameters

    Parameter Description
    <show_number> Will show all rounds/questions from a show.
    <show_number> / round number (1-3) Will show all questions from a selected round of a show.
    date / <year> Will return all shows from a given year.
    date / <year> / <month> Will return all shows from a given year/month.
